# Contributing to NasuaOS

First of all, thank you for considering contributing to NasuaOS!

## Before you start

- Fork the repository.
- Create a new branch from `main`.
- Keep commits focused on a single change.
- Make sure your code builds successfully.

## Coding Style

- Use C++17.
- Follow the existing formatting style.
- Use meaningful variable and function names.
- Comment complex code where necessary.


## Pull Requests

Please include:

- A clear description of the change.
- Screenshots (if UI related).
- Testing information.
- Related issues (if applicable).

## Reporting Bugs

Please include:

- NasuaOS version
- Emulator or hardware
- Steps to reproduce
- Expected behavior
- Actual behavior
- Logs or screenshots

## Feature Requests

Explain:

- What problem it solves.
- Why it would be useful.
- Possible implementation ideas.

Thank you for helping improve NasuaOS!
